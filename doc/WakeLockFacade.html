A facade exposing some of the functionality of the PowerManager, in particular wake locks.
@author Felix Arends (felixarends@gmail.com)
@author Damon Kohler (damonkohler@gmail.com)
wakeLockAcquireBrightAcquires a bright wake lock (CPU on, screen bright).
wakeLockAcquireDimAcquires a dim wake lock (CPU on, screen dim).
wakeLockAcquireFullAcquires a full wake lock (CPU on, screen bright, keyboard bright).
wakeLockAcquirePartialAcquires a partial wake lock (CPU on).
wakeLockReleaseReleases the wake lock.
